# HEARTBEAT.md — 小龙虾心跳自检

## 每次心跳必须检查

### 1. TASKS.md P0 检查
- [ ] 有没有 P0 未完成承诺？
- [ ] 有没有 PENDING 超过24小时未处理的？
- 如有 → **立即处理，不做新任务**

### 2. pending_verification 检查
- [ ] 有没有标记为"未验证"的项？
- [ ] Telegram 通道是否畅通？
- 如有 → **立即处理或上报**

### 3. 通道回复检查
- [ ] 本轮有无 Telegram/企微/QQ 用户在等回复？
- [ ] 我是否通过正确通道回复了？
- 如有遗漏 → **立即补救**

### 4. 自审三问
- [ ] 今天说了多少承诺？有没有漏写的？
- [ ] 今天犯了几次铁律违规？
- [ ] 有没有悬空操作（做了但没写文件）？

---

## 通过条件

以上全部通过 → `HEARTBEAT_OK`

有 P0 悬空 or 通道遗漏 → **立即预警用户，不发 HEARTBEAT_OK**
