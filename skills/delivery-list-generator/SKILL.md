# 发货清单生成器

## 输入信息

用户需要提供：
- 定制方（客户单位全称）
- 订单编号
- 承揽方（我方公司名，默认：醴陵韶峰服饰销售有限公司）
- 发货日期
- 商品明细列表（序号、物料编码、物料名称、物料描述、单位、数量）

## 生成模板字段

| 字段 | 说明 |
|------|------|
| 发货清单 | 标题 |
| 定制方 | 客户单位全称 |
| 订单编号 | 客户提供的订单号 |
| 序号 | 自动编号 |
| 物料编码 | 商品编码 |
| 物料名称 | 商品名称 |
| 物料描述 | 规格型号颜色等 |
| 单位 | 件/套/个等 |
| 数量 | 数字 |
| 备注 | 可空 |
| 承揽方 | 我方公司名 |
| 发货日期 | 日期 |

## 输出格式

- xlsx文件
- 专业字体（Arial）
- 黄色/蓝色表头区分
- 商品行白色底
- 合计行高亮
- 打印区域设置
