# client-stage-tracker

政府 B2G 客户阶段追踪器。

## 触发

`阶段：[客户名] + [当前情况描述]`

例如：`阶段：上海市教育局 + 刚拿到局长电话，还没见过面`

---

## 阶段识别逻辑

根据描述判断所处阶段：

| 阶段 | 判断依据 |
|---|---|
| **S0 还没开始接触** | 刚拿到线索/电话，没有建立任何联系 |
| **S1 准备首次拜访** | 约到了拜访时间，还没见面 |
| **S2 第一次见面** | 今天/明天/最近要第一次见面 |
| **S3 挖掘真实需求** | 已经见过面，想确认真实需求 |
| **S4 方案推进中** | 在讨论方案/报价/采购流程 |
| **S5 临门一脚** | 快要签约/卡在最后审批/拖了很久 |

---

## 阶段输出模板

**注意：每个阶段只输出一个最高价值动作，不是动作列表。**

---

### S0 → 还没开始接触

**最高价值动作：** 快速情报扫描（20分钟）
在见面前搞清楚：考核压力是什么 / 预算大概多少 / 有没有竞争对手在谈

**最可能的隐形障碍：** 没有信任基础，第一句话就暴露目的，被当推销员

**48小时具体下一步：**
1. 搜索「[部门] 政府工作报告 2026 重点任务」
2. 找到1个能用的破冰话题（政策依据或热点事件）
3. 准备一句不暴露目的的开场白

---

### S1 → 准备首次拜访

**最高价值动作：** 定制开场白（用他说过的话，不用产品词）

**最可能的隐形障碍：** 见面时不知道先说什么，被牵着鼻子走

**48小时具体下一步：**
1. 搜索该领导最近的公开发言（局长/主任 讲话 2025 2026）
2. 找到1句原话，作为开场引用
3. 准备一个和产品无关的破冰话题（从招聘信息/政府报告推断）

---

### S2 → 第一次见面

**最高价值动作：** 听而不是说——用一个问题让对方说更多

**最可能的隐形障碍：** 说得太多，暴露方案，被对方拿去比价

**48小时具体下一步：**
1. 想好一个追问：「你们现在遇到的最大挑战是什么？」
2. 准备好被问到自己时的回答（30秒，不超过1分钟）
3. 不带产品册，带案例一页纸 + 政策依据

---

### S3 → 挖掘真实需求

**最高价值动作：** 识别「行为矛盾」——他说的 vs 他实际关心的

**最可能的隐形障碍：** 他表面说A，实际顾虑是B（通常是风险/预算/上级看法）

**48小时具体下一步：**
1. 回顾见面时他哪些话题说得多、哪些避开了
2. 针对避开的那个话题，准备追问：「这方面最近有没有什么新要求？」
3. 判断他的个人动机（安全导向/政绩导向/业务导向）

---

### S4 → 方案推进中

**最高价值动作：** 给他准备一份「向上汇报的一页纸」

**最可能的隐形障碍：** 他内部汇报时说不清楚，你的方案被上级否决

**48小时具体下一步：**
1. 写出「问题背景 + 政策依据 + 方案概述 + 预期成果」的一页纸
2. 预判隐形否决者（纪检监察/财务/法制科），提前准备应对
3. 问他：「最近上级有没有什么新的时间节点要求？」

---

### S5 → 临门一脚

**最高价值动作：** 找出隐形否决者 + 给他一个现在就签的理由

**最可能的隐形障碍：** 不是他不想签，是他背后有人还没点头

**48小时具体下一步：**
1. 直接问他：「合同审批还有什么卡点吗？」
2. 如果有顾虑，写一份书面承诺（验收标准 / 达不到就退款）
3. 给他时间压力：「X检查是X月，采购流程需要X个月，现在启动刚好」

---

## 客户记录更新（mem0ai）

每次触发后，立即更新客户档案：

1. 用 `memory_search` 搜索该客户名是否已有记录
2. 若有：在对应 memory 文件中更新阶段 + 本次关键洞察
3. 若无：创建 `memory/YYYY-MM-DD-client-[客户名].md`

档案格式：
```markdown
# [客户名] 阶段追踪 · [日期]

## 当前阶段：S[0-5] · [阶段名]
## 上次更新：[日期]

## 关键记录
- [日期] [阶段]：[一句描述当前状态/洞察]

## 阶段历史
S0 → S1 → S2 → ...（时间线）

## 待处理
- ___
```

---

## 输出格式（每次触发）

```
**📍 [客户名] · S[0-5] · [阶段名]**

🎯 最高价值动作：[一个具体动作，不是列表]

⚠️ 隐形障碍：[他不说但真实存在的顾虑]

📌 48小时下一步：
1. [具体可执行的动作]
2. [备用/强化]

📁 档案已更新 → [memory 文件路径]
```
