# Gemini 协作代理

## 核心方法论

### 三步法则（最重要！）
**观察 → 思考 → 行动**

每次操作前都要：
1. **观察**：截图看当前界面状态
2. **思考**：想清楚要做什么，会遇到什么
3. **行动**：执行，不要盲目试探

### 操作前自检
- [ ] 只用一个对话窗口
- [ ] 等待上一轮回复完整
- [ ] 打字间隔随机
- [ ] 失败就停，不超过3次尝试

## Gemini 对话管理

### 打开对话
```
左侧栏 → 点击 "Gem" 或 "对话"
```

### 清理对话历史（重要！避免被检测）

**操作步骤：**
1. 左侧栏找到要删除的对话
2. 鼠标移到对话项上 → 出现三个竖点（•••）
3. 点击三个竖点 → 弹出菜单
4. 菜单选项（共4个）：
   - 分享对话内容
   - 固定
   - 重命名
   - **删除**（最后一个）
5. 点击"删除" → 对话消失

**删除按钮位置：** 竖点菜单的第4个选项

### 选择器
| 元素 | 选择器 |
|------|--------|
| 对话列表 | `.conversation-items-container` |
| 三个竖点 | 鼠标hover后出现 |
| 删除菜单项 | 竖点菜单第4项 |

## 通过观察学习

### 观察用户操作流程
1. 截当前界面图
2. 描述观察到的变化
3. 总结用户操作步骤
4. 记住操作流程

### 关键原则
- 用户操作时 → 截图记录
- 完成后 → 确认是否成功
- 失败 → 问用户哪里做错了
