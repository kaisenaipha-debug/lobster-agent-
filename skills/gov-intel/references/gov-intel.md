# 政府情报完整协议

## 核心认知

政府客户决策驱动因素：政策合规 + 个人仕途安全 + 上级考核
不是「我们很优秀」，是「选我们是安全的」。

---

## 冷情报协议（第一次拜访前，零接触情报）

触发格式：
「冷情报：[城市名] [部门名]
 我的产品/服务方向：[一句话]
 计划拜访时间：[大概什么时候]」

### 五个动作

**动作一：政府工作报告**
搜索：[城市名] [年份] 政府工作报告 [部门关键词]
提取：今年被点名的具体任务 / 考核指标 / 时间节点
输出：考核压力 / 政策窗口 / 领导关注点

**动作二：招聘信息**
搜索：[部门名] 招聘 2025 2026（Boss直聘/猎聘/官网）
提取：招什么职位 / 暴露什么战略方向 / 预算灵活性
输出：战略信号 / 预算信号 / 切入点

**动作三：采购记录**
搜索：[城市名] [部门名] 采购 中标 招标 2024 2025
提取：买过什么 / 哪家中标 / 金额范围
输出：竞争态势 / 采购方式预判 / 竞争对手清单

**动作四：领导公开发言**
搜索：[部门名] 局长/主任 发言 讲话 2025
提取：高频关键词 / 强调重点 / 语言风格
输出：破冰话题 / 他的语言（见面时用他的词）/ JTBD推断

**动作五：同类城市案例**
搜索：[省内/全国] [部门名] [产品方向] 案例 2024 2025
提取：哪个城市做了 / 成果数字 / 项目规模
输出：最强案例（带数字）/ 见面时的用法

### 最终输出：第一次拜访准备包

- **情报摘要**：部门画像3句 / 领导画像2句 / 时机判断
- **开场白**：用他说过的话，不介绍自己，不说产品
- **破冰话题**
- **过渡句**：从破冰到业务
- **携带材料**：案例一页纸 + 政策依据，不带产品册
- **信息缺口**：见面时最值得追问的一个问题
- **风险提示**（红队）：最可能失败的原因

每条情报标注：✅已确认 / ⚠️推断(置信度X%) / ❓未知

## 五层深度情报框架

### L1 · 政策与考核层（最高价值）
- 当前政策背景：国家/省/市三级政策文件
- 年度考核压力：核心KPI / 上级检查什么 / 节点
- 预算周期：3-4月确定 / 年底突击花钱 / 专项资金

### L2 · 决策结构（政府特殊版）
- 形式决策者 vs 实质决策者
- 三个关键角色：推动者 / 守门人 / 否决者
- 采购方式：<50万单一来源 / 50-200万竞争性谈判 / >200万招标

### L3 · 真实痛点（藏在行为里不在语言里）
- 表层说法 vs 真实痛点对照表
- 官员个人动机：做了不出事 > 做好了有功劳
- 前车之鉴：上一个供应商为什么没续约

### L4 · 时机与触发器
- **高价值时机**：1-3月（政策落地）/ 4-6月（上半年考核前）/ 9-10月（预算申报）/ 11-12月（年底花钱）
- **触发事件**：领导换届 / 专项检查通知 / 负面曝光 / 兄弟城市做了同类项目 / 专项资金下拨

### L5 · 关系地图
- 三类有价值的关系：上级关系 / 平级背书 / 内部推动者
- 背书权重：上级领导 > 本市案例 > 兄弟部门 > 行业专家 > 自己说
- 最高效路径：已合作部门 → 做出成果 → 系统内部介绍

## 信任转化五个动作

1. **先说政策依据再说产品**
   「根据XX部委X号文件要求，这个方向上级明确要推...」
   让他听到：做这件事符合政策，不做才有风险

2. **拿出同级成功案例**
   「XX市XX局去年做了同类项目，结果是...」
   政府官员最怕第一个吃螃蟹

3. **给他一份「向上汇报」的一页纸**
   问题背景 + 政策依据 + 方案概述 + 预期成果
   让他拿去汇报，不是他自己解释

4. **明确书面承诺**
   写清楚验收标准，达不到全额退款/免费整改
   书面承诺是他的免责盾牌

5. **永远不要催**
   替代动作：「最近上级有没有新的时间节点要求？我这边可以配合你们的节奏」

---

## 成交周期缩短三个动作

给他「现在推」的三个理由：
- **时间压力**：「XX检查是X月，走采购流程需要X个月，现在启动刚好」
- **政策窗口**：「这个专项资金申报截止是X月，过了要等明年」
- **竞争压力**：「隔壁XX局已经在做了，你们第一个还是跟随，对年底汇报分量不一样」

## 隐形否决者必查五个角色

1. 纪检监察
2. 分管财务副职
3. 法制科
4. 上级主管部门
5. 内部竞争者

## 政府官员 JTBD 五种类型

| 类型 | 雇用目标 | 语言策略 |
|---|---|---|
| 完成上级指标 | 用考核语言，说清楚能帮他交差 | KPI导向 |
| 任期内留下痕迹 | 强调项目的标志性和长期价值 | 政绩导向 |
| 规避风险安全落地 | 强调合规、先例、书面承诺 | 安全导向 |
| 证明自己判断力 | 给他一个可以展示的故事 | 故事导向 |
| 解决真实业务痛点 | 直接说能解决什么，怎么衡量 | 业务导向 |
