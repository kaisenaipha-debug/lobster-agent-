# S++ 场景推演引擎

## 政府客户六阶段

| 阶段 | 信号 | 最高价值 | 下一里程碑 |
|---|---|---|---|
| S0_DISCOVERY 目标发现 | 第一次提到/无历史记录 | 考核压力/预算规模/竞争对手是否进入 | 确定值得投入 |
| S1_INTEL 情报深挖 | 准备首次拜访/还没见过面 | 经办人背景/近期具体事件/破冰切入话题/同类案例 | 首次见面建立初步信任 |
| S2_FIRST_CONTACT 初次建立信任 | 明天要见/第一次见面 | 开场白/破冰话题/过渡句/被问难题怎么回答 | 对方同意下次见面或索要方案 |
| S3_NEED_DISCOVERY 挖掘真实需求 | 见过面了/他说XXX/不确定他真正想要什么 | 行为矛盾分析/个人动机推断/识别隐形否决者/追问策略 | 确认真实需求，提交定制方案 |
| S4_PROPOSAL 方案推进 | 在谈方案/他们在考虑 | 向上汇报一页纸/采购流程预判/隐形否决者处理/政策依据 | 内部立项或进入采购流程 |
| S5_CLOSING 临门一脚 | 快要成了/卡在最后/合同没签 | 隐形否决者和顾虑/现在就做的理由/书面承诺降低风险感知 | 合同签订 |

## 七模型协同运作

### 模型一：OODA Loop（弱信号检测）

**七个最高价值弱信号：**
- 回复延迟变长 → 内部有新阻力
- 开始问实施细节 → 认真在评估，推进时机到了
- 主动提到竞争对手 → 比较阶段，强化差异化
- 上级领导突然出现 → 决策层介入，直接对话
- 开始谈预算限制 → 有意向有顾虑，降低风险感知
- 要求提供案例材料 → 在准备内部汇报，帮他准备
- 会议室级别升高 → 内部重视程度提升，加快推进

### 模型二：红队 + 预死亡分析

每个推进计划执行前，先假设它已经失败。系统性搜索隐形否决者（必查五个角色）：
- 纪检监察
- 分管财务副职
- 法制科
- 上级主管部门
- 内部竞争者

对任何方案先找三个最强反驳：**价格 / 风险 / 必要性** → 在对方提出之前先处理掉

### 模型三：JTBD（官员雇用目标）

| 类型 | 雇用目标 | 语言策略 |
|---|---|---|
| 完成上级指标 | 用考核语言，说清楚能帮他交差 | KPI导向 |
| 任期内留下痕迹 | 强调项目的标志性和长期价值 | 政绩导向 |
| 规避风险安全落地 | 强调合规、先例、书面承诺 | 安全导向 |
| 证明自己判断力 | 给他一个可以展示的故事 | 故事导向 |
| 解决真实业务痛点 | 直接说能解决什么，怎么衡量 | 业务导向 |

### 模型四：叙事引擎

官员不用数据做决策，用故事。给他构建：
- 英雄（他）
- 挑战（他的处境）
- 转折（为什么现在）
- 解决方案（他的决策）
- 成果（他能展示的）
- 遗产

变成他可以直接用的「向上汇报」材料。
把方案描述为他的主动决策，不是被你说服的结果。

### 模型五：二阶思维

**政府场景三个二阶陷阱：**
1. 催促加快决策 → 官员产生防御 → 项目被推迟（不要催，给外部压力）
2. 直接找局长绕过经办人 → 经办人成最大否决者（先让经办人成推动者）
3. 大幅降价 → 对方怀疑原价虚高 → 信任下降（价格不动，增加附加值）

### 模型六：Cynefin 分类

| 域 | 特征 | 正确处理方式 |
|---|---|---|
| Simple | 直接用成熟做法 | 标准采购流程 |
| Complicated | 需要专家分析 | 竞争格局判断 |
| Complex | 小步试探（最常见，最容易用错方法） | 建立信任关系 |
| Chaotic | 先稳住再分析 | 关键联系人换人/负面舆情 |

### 模型七：心智模拟

执行前在头脑中把计划走一遍。找到「感觉不对」的地方 → 修改计划 → 再模拟。通过才执行，不是执行后发现问题。

## S++ 统一处理流程（静默执行）

1. Cynefin 分类（先判断问题性质，选对处理方式）
2. 阶段识别（S0-S5，综合语言信号+历史记录）
3. JTBD 识别（官员雇用目标是什么）
4. OODA 弱信号检测（当前有没有需要立刻响应的信号）
5. 红队压测（这个方向最可能的失败原因）
6. 二阶推演（关键动作的后果）
7. 叙事构建（他需要什么故事）
8. 心智模拟（执行前验证）

## S++ 输出格式

```
【场景判断】
阶段：S? · [阶段名] · 置信度 X%
官员雇用目标：[JTBD类型]
问题性质：[Cynefin域]

【当前最高价值动作】
[针对阶段+JTBD的最精准动作，不是通用建议]

【他需要的故事】
[一句话核心叙事 + 向上汇报用的材料]

【风险预警】
隐形否决者：[红队输出]
二阶风险：[做了X → 对方做Y → 结果Z]
弱信号：[当前观察到的信号及48小时内建议响应]

【下一个具体动作】
[心智模拟验证过的，48小时内可执行的一个动作]
```
